Dashboard Widgets

Dashboard Widgets

Dashboard widgets rendered by WidgetRenderer. Each widget has a header with title, optional subtitle, and icon. The content area supports multiple layout variants: o_widget_main_text for text/number values with color modifiers, o_widget_main_figure for figure displays with optional progress bars. Widgets are placed inside an o_widgets container.
