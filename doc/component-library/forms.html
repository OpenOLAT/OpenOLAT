Text Inputs

Text Input (Horizontal Layout)

Standard text input rendered in the horizontal form layout with label in col-sm-3 and input in col-sm-9. This is the default form layout used throughout OpenOlat.

o_form form-horizontal form-group form-control control-label col-sm-3 col-sm-9
Bitte geben Sie eine gueltige E-Mail Adresse ein.
Show Code 
<form class="o_form form-horizontal">
  <div class="form-group">
    <label class="col-sm-3 control-label">Vorname</label>
    <div class="col-sm-9">
      <input type="text" class="form-control" placeholder="Max">
    </div>
  </div>
  <div class="form-group">
    <label class="col-sm-3 control-label">Nachname</label>
    <div class="col-sm-9">
      <input type="text" class="form-control" placeholder="Mustermann">
    </div>
  </div>

      <span class="help-block">Bitte geben Sie eine gueltige E-Mail Adresse ein.</span>
    </div>
  </div>
</form>

Error State

Validation errors are shown by adding has-error to the form-group. The error message is displayed in an o_error div below the input.

has-error o_error o_icon_mandatory
Dieses Feld ist erforderlich.
Bitte geben Sie eine gueltige E-Mail Adresse ein.
Show Code 
<form class="o_form form-horizontal">
  <!-- Error state -->
  <div class="form-group has-error">
    <label class="col-sm-3 control-label">
      <i class="o_icon o_icon_mandatory"></i> Kurstitel
    </label>
    <div class="col-sm-9">
      <input type="text" class="form-control" value="">
      <div class="o_error">Dieses Feld ist erforderlich.</div>
    </div>
  </div>

  <!-- Success state -->
  <div class="form-group has-success">
    <label class="col-sm-3 control-label">Benutzername</label>
    <div class="col-sm-9">
      <input type="text" class="form-control" value="mmustermann">
    </div>
  </div>
</form>

Tag Input (TextBoxList)

Tag Input (Tagify)

Tag input field using the Tagify library. Tags are displayed as pill-shaped elements with the o_tag class. The container uses the o_textbox class for styling consistent with standard form inputs. Tags can be selected (o_tag_selected) or clickable (o_tag_clickable).

tagify o_textbox o_tag o_tag_selected o_tag_clickable
Mathematik Grundlagen Semester 1
Physik Chemie Mathematik Biologie Informatik
Show Code 
<!-- Tagify input (rendered by JS) -->
<tags class="tagify o_textbox">
  <tag><div><span class="tagify__tag-text">Mathematik</span></div>
    <span class="tagify__tag__removeBtn"></span>
  </tag>
  <span class="tagify__input" contenteditable></span>
</tags>

<!-- Tag selection (clickable tags) -->
<a href="#" class="o_tag o_tag_clickable">Physik</a>
<a href="#" class="o_tag o_tag_selected o_tag_clickable">Mathematik</a>

Help Tooltip

Context Help (chelp)

A small question-mark icon that links to the manual or shows a tooltip with help text. The o_chelp class styles the icon as a bordered circle. When used inline in forms, o_form_chelp positions it next to the label. The o_chelp_wrapper provides a float-right container for page-level help.

o_chelp o_chelp_wrapper o_form_chelp o_chelp_tooltip o_icon_help
Standalone help link: Hilfe
Tooltip icon (in form label):
Show Code 
<!-- Standalone help link -->
<span class="o_chelp_wrapper">
  <a href="manual_url" class="o_chelp" target="_blank">
    <i class="o_icon o_icon_help"></i> Hilfe
  </a>
</span>

<!-- Inline tooltip icon -->
<i class="o_chelp_tooltip o_icon o_icon-fw o_icon_help"
   title="Geben Sie den vollstaendigen Kurstitel ein."></i>

<!-- Help icon in form label -->
<label class="col-sm-3 control-label">
  <i class="o_icon o_icon_mandatory"></i> Kurstitel
  <i class="o_chelp_tooltip o_form_chelp o_icon o_icon-fw o_icon_help"
     title="Der Kurstitel wird in der Navigation angezeigt."></i>
</label>

Form Layout

Standard Horizontal Form Layout

The standard OpenOlat form layout uses form-horizontal with labels in col-sm-3 and inputs in col-sm-9. A form submit row offsets the buttons with col-sm-offset-3. Spacers (o_spacer) separate form sections.

o_form form-horizontal form-group col-sm-3 col-sm-9 col-sm-offset-3 o_spacer
Speichern Abbrechen
Show Code

    <label class="col-sm-3 control-label">
      <i class="o_icon o_icon_mandatory"></i> Vorname

      <input type="text" class="form-control" value="Max">
    </div>
  </div>

  <hr class="o_spacer">

  <div class="form-group">
