The Paella Player

Проигрыватель Paella предназначен для воспроизведения двух видеопотоков, с камеры и экрана преподавателя, а так же дополнительной информации о лекции. Навигация по лекции доступна по слайдам и закладкам. Для воспроизведения лекции сначала можете нажать кнопку "воспроизведение" в окне с видео или на панели управления.


Play & Pause

Кнопки для старта воспроизведенеия и паузы. Если щёлкнуть левой кнопкой мыши за пределами панели, то воспроизведение так же начнётся или поставится на паузу.


Volume

Управление громкостью воспроизведения с помощью всплывающего окна.


Back & Advance

Кнопки для перемотки вперёд/назад.


Playback Rate

С помощью этого элемента управления пользователи могут изменить скорость воспроизведения.


Qualities

Выбрать разрешение видео.


Tabs

Дополнительные модули, позволяющие пользователям оставлять комментарии и просмотривать дополнительную информацию о видео.


Skin

Выбор темы оформления проигрывателя.


Social

Поделиться видео в Facebook, Twitter или получить ссылку для встраивания видео в HTML страницы.


Fullscreen

Кнопка переключения между полноэкранным и оконным режимом.


Player View Mode

Используйте эту кнопку для изменения размера источников или для изменения взаимного расположения видео.


Captions

Включение субтитров для загруженного видео. Когда активировано, пользователь может выбирать между различными языками либо отключать субтитры.


Frame Control

Использование этих кнопок позволяет показать слайды из видео. Выбор слайда с помощью мыши позволяет перейсти к месту в видео, где появляется этот слайд.


Search

Пользователи могут производить поиск по по всем источникам информации о видео (распознанный текст со слайдов, субтитры). Результат будет показан во всплывающем окне с сортировкой по времени появления в видео.


Rate

Используёте эту кнопку для просмотра рейтинга и оценки этого видео.
