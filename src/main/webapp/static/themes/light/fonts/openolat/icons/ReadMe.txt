Preparing SVG logos for the OpenOlat font
=========================================

- Open the SVG file in Inkscape
- Remove unwanted stuff e.g. product name, slogans
- Resize document to content
    - Go to File â†’ Document Properties... (or press Shift + Ctrl + D )
    - In the Custom size area on the Page tab, expand Resize Page to content... (or press Alt + Z )
    - Click the Resize page to drawing or selection button (or press Alt + R )
- Save as plain SVG

The SVG logo is now ready for usage in fontello.
