## ============================================================================ ## AI Questions submit hook — always rendered, never hidden. ## ## On form submit (before the AJAX round-trip completes) this script does ## two things in one synchronous tick: ## 1. Suppresses the global window busy spinner for exactly one round-trip ## so no grey full-screen overlay flashes before the pulse appears. ## 2. Adds `o_ai_questions_generating` to the form's root fieldset so ## the CSS overlay becomes visible *instantly* — the user sees the ## pulse the moment they click the submit button. ## ## Both suppressions are lifted by the overlay script once the AJAX response ## is applied and the overlay fragment is in the DOM. ## ============================================================================
